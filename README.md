# customer_behaviour_analysis
Data analytics project showcasing customer behavior analysis using python, Sql, and power BI 
📊 Data Behavior Analysis Project
1. Overview

This project focuses on analyzing customer behavior using a complete data analytics workflow. The analysis involves loading the dataset in Python, performing Exploratory Data Analysis (EDA), cleaning the data, running analytical queries in MySQL, and building an interactive dashboard in Power BI.

The goal of this project is to extract meaningful insights about customer behavior, purchasing patterns, and business performance to support data-driven decision making.

2. Dataset
The dataset used in this project contains information related to customer transactions and purchasing behavior.
Typical columns include:

Customer ID

Age / Demographics

Product Category

Purchase Amount

Purchase Date

Subscription Status

Previous Purchases

The dataset was first loaded into Python for preprocessing and later stored in MySQL for querying and analysis.

3. Tools & Technologies
The following tools were used throughout the project:
Python –
 Data loading, cleaning, and exploratory data analysis

Pandas & NumPy – Data manipulation and preprocessing

Matplotlib / Seaborn – Data visualization during EDA

MySQL – Writing SQL queries for deeper data analysis

Power BI – Creating an interactive dashboard

Gamma AI – Generating a professional presentation (PPT)

5. Project Steps
Step 1: Data Loading

Imported the dataset into Python (Jupyter Notebook)
Loaded data using the Pandas library

Step 2: Exploratory Data Analysis (EDA)

Checked dataset structure and summary statistics

Identified missing values and duplicates

Analyzed distributions and relationships between variables


Step 3: Data Cleaning

Removed duplicates

Handled missing values

Converted incorrect data types

Standardized column formats

Step 4: SQL Analysis (MySQL)

The cleaned dataset was imported into MySQL for query-based analysis such as:

Customer purchase frequency

Category-wise sales analysis

Customer segmentation

Repeat customer identification

Step 5: Dashboard Creation

Connected Power BI with the cleaned dataset and built an interactive dashboard to visualize:

Sales trends

Customer behavior patterns

Category-wise performance

Key business KPIs

Step 6: Reporting

Created a Power BI report summarizing key insights
Generated a presentation using Gamma AI to communicate findings clearly.

5. Dashboard
The Power BI dashboard includes:

Sales overview and trends

Customer purchase behavior

Product category performance

Customer segmentation insights

Key metrics and KPIs

The dashboard allows users to interactively explore the data and identify business opportunities.

6. Key Results / Insights

Some of the key insights from the analysis include:

Identification of high-value customer segments

Understanding of most purchased product categories

Detection of repeat buying behavior

Insights into customer purchase trends over time

These insights can help businesses improve marketing strategies, product offerings, and customer engagement.
